# Week 2 - Student Grade Calculator

## 📌 Project Overview
This project is a Python program that calculates a student's grade based on marks entered (0–100).  
It teaches decision-making, loops, functions, and input validation.

---

## 🎯 Objectives
- Use if–elif–else statements
- Validate user input using while loops
- Write reusable functions
- Display helpful messages
- Build an interactive Python program

---

## 🛠 Setup & Installation
1. Install Python (if not installed)  
   https://www.python.org
2. Save the file `grade_calculator.py'
3. Run the program:



python grade_calculator.py

---

## 🔍 Code Structure

- **calculate_grade(marks)**  
  Function that returns grade + message
- **While loop**  
  Ensures marks are between 0 and 100
- **If–elif–else**  
  Grade logic
- **Input/Output**  
  Interactive program for any student

---

## 🧠 Grading Logic
- A → 90–100  
- B → 80–89  
- C → 70–79  
- D → 60–69  
- F → 0–59
